# Sesame Credit Quiz

A humorous, interactive social credit score quiz built with Python and Streamlit. Inspired by China's Social Credit System, this app asks users ten lighthearted questions about their financial habits, social behavior, community involvement, and personal style ‚Äî then calculates a satirical "Social Credit Score" between 350 and 950.

Based on their score, users are awarded a tier (VIP, Gold, Silver, or Bronze) along with a set of absurd but entertaining social perks ‚Äî everything from "red carpet service at coffee shops" to "honorary title as the Mogul of Mingling."

## How It Works

- Users answer 10 multiple-choice questions, each scored on a scale of 0‚Äì20 points
- A total raw score is calculated and mapped to a predicted Social Credit Score (350‚Äì950)
- The score determines a tier and a corresponding set of humorous social benefits
- Users can retake the quiz at any time

1. Install dependencies: `pip install streamlit`
2. Run the app: `streamlit run streamlit_app.py`
3. Open your browser to the local Streamlit URL (typically `http://localhost:8501`)
